## 📞 If You Have Issues

### Login button doesn't show
- Check: Does `<div id="auth-container"></div>` exist in HTML? (After header)
- Check: Is the module import script present before `</head>`?

### "FirebaseModule is not defined"
- Check: Is the module import correct? `import * as FirebaseModule from '/static/js/firebase-integration.js'`
- Check: Are you using `type="module"` in the script tag?

### Cart doesn't save to Firebase
- Check: Are you logged in? (Should show avatar)
- Check: Is `window.FirebaseModule` defined?
- Check: Browser console (F12) for errors

### Payment modal doesn't open
- Check: Does `<div id="payment-modal-container"></div>` exist?
- Check: Is payment-modal.js loaded?
- Check: Browser console for errors

### Still having issues?
- Check the documentation: FRIEND_CODE_INTEGRATION_GUIDE.md
- Review the code in firebase-integration.js (well-commented)
- Check browser console (F12 → Console tab) for error messages
